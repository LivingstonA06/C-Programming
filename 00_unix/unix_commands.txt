mkdir <folder name> - make directory

ls - list. Displays the content in the directory.

cd <folder name> - change directory. Move to another directory.

code <file name> - creates a file and opens it in a text editor.

pwd - print working directory. shows which file you are in

gcc <file name (source code)> - compiler.

./a.out - run the source code

./ - run the file in the directory.

\- - escape character.

\* - escape sequence.

\n - new line.

Up Key - cycle through previously used commands.

CTRL J - hide and unhide terminal.

CTRL B - hide and unhide the explorer
